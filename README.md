## üöÄ Getting Started

### 1. Installation
Sideload the extension directly into your Antigravity IDE:
```bash
npm install
npm run sideload
```

### 2. Configuration
Create a `.env` file in the root directory:
```env
OPENROUTER_API_KEY=your_key_here
```

### 3. Usage
- **Start the Bridge**: Use the command `depthos-bridge.start` from the Command Palette (`Cmd+Shift+P`).
- **Query the Ensemble**: Interact via the `bridge_query` tool. The bridge will assemble a team, inject the Constitutional Core, and synthesize a grounded response.
- **Ratify Actions**: Use `bridge_ratify` to exercise your **Human Veto** over proposed execution steps.
